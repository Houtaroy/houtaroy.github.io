# 运算

## 加倍

左移一位相当于乘2

```java
int a = 12;
b = a << 1;
// b = 12 * 2
// b = 24
```

## 减半

右移一位相当于除以2

```java
int a = 9;
b = a >> 1;
// b = 9 / 2
// b = 4
```

## 判断奇偶

二进制最后一位为1则为奇数

```java
public boolean isEven(long a) {
    return (a & 1) == 0;
}
```

## 交换变量

异或操作有两个法则:

- 异或自身等于0
- 异或0等于自身

```java
int a = 12;
int b = 9;

a ^= b;
b ^= a;
a ^= b;

// a = (12 ^ 9)
a = a ^ b;
// b = (12 ^ 9) ^ 9
// b = 12 ^ 9 ^ 9
// b = 12
b = (a ^ b) ^ b
// a = (12 ^ 9) ^ 12
// a = 12 ^ 12 ^ 9
// a = 9
a = (a ^ b) ^ a
```
